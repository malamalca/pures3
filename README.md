Projekt je PHP aplikacija za izračun gradbene fizike po PURES 3 pravilniku.

Trenutno je implementirana gradbena fizika, sistemi TSS razsvetljava, TSS prezračevanje.

V delu pa je še sistem TSS ogrevanje.
